# CLAUDE.md

Memoria e instrucciones del proyecto para asistentes de IA.

## Estructura
- `docs/`: arquitectura, decisiones y runbooks
- `.claude/`: configuración local, hooks y skills
- `tools/`: scripts y prompts de trabajo
- `src/`: código fuente principal de la aplicación

## Contexto rápido
Aplicación de historia clínica con Next.js, Drizzle ORM y despliegue en Cloudflare.
